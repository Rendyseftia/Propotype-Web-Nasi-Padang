**Ringkasan Proyek:**
Aplikasi web *full-stack* yang dirancang khusus untuk mendigitalisasi operasional Rumah Makan Padang. Sistem ini mengubah proses reservasi manual menjadi sistem terpusat yang menangani pemilihan meja visual, pre-order menu, hingga pelaporan keuangan secara *real-time*.

**Masalah yang Diselesaikan:**
* Mengatasi inefisiensi pemesanan manual via WhatsApp yang rawan kesalahan (*human error*).
* Mencegah masalah *double-booking* (dua pelanggan memesan meja yang sama).
* Mengurangi waktu tunggu pelanggan di restoran (*table turnover rate*) karena menu sudah dipesan sebelumnya.

**Fitur Utama:**
* **Visual Seat Management:** Pelanggan memilih meja secara interaktif dengan indikator status *real-time* (Available, Booked, Walk-in).
* **Smart Order System:** Pre-order menu dengan kalkulasi harga otomatis.
* **Auto-Cancel Timer:** Sistem timer pembayaran 5 menit yang terintegrasi dengan database untuk mencegah *ghost booking*.
* **Real-time Admin Dashboard:** Monitoring pesanan masuk tanpa refresh halaman (menggunakan AJAX Polling) dan rekap pendapatan harian terpisah (Cash vs Transfer).
* **Invoice Generator:** Pencetakan bukti bayar digital (PDF) otomatis.
